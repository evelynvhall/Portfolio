Backing Up Your Data

As with all software, it is important to keep IMPS backed up regularly.

Backing up photos

IMPS can create an archive of all the image files in the item image directory. When you installed or configured IMPS, you set the path to the image storage directory. All image files in this directory can be archived and downloaded from the control panel.

To back up your item photos:

  1. From the IMPS home page, click the Control panel (CP) icon
    ().
  2. Select the Backups tab in the control panel.
  3. Click the Archive photos button.
  4. Once the archive is created, IMPS will show the download page. You can click the Download button to download the archive onto your local computer.

    5. Note: The archive contains all the image files you have stored on IMPS. While only the most recent archive can be downloaded from the IMPS control panel, IMPS does not delete archive files from the disk. You can find all your old archive files in the directory shown in the Backup directory field on Server info tab on the Control panel.

Backing up the database

IMPS has a built-in ability to create a backup of the database. To start a database backup:

  1. From the IMPS home page, click the Control Panel (CP) icon
    ()
  2. Click Backup now to create a backup.
  3. Once the data backup file has been created, IMPS will show the download page. You can click the Download button to download the backup file to your local computer.

    4. Note: The backup contains all the necessary SQL to re-create your database. While only the most recent backup can be downloaded from the IMPS control panel, IMPS does not delete backup files from the disk. You can find all your old backups in the directory shown in the Backup directory field on Server info tab on the Control panel.
