Using locations

About locations

IMPS locations are arbitrary groups that you can assign to boxes which provide an additional tool to further organize, sort, and find them. By default, new boxes are assigned to the "Unspecified" location.

Locations apply to boxes, not items. The "By loc" button on the home page shows a list of boxes.

Assigning a locations is optional. When you add or edit an box, you can:

  • Choose not to select a location, which means the box location will set to "Unspecified"
  • Select an existing location to assign the box to
  • Type a new location name for this box. The new location will be used for this box and will be added to the list of locations in the location selection field when you use it in the future.

Managing locations

The easiest way to work with locations is to select or add them when adding boxs. However, you can add, edit, or delete locations using the control panel.

Adding a location

If you add a new location using the control panel, it will be added to the list of locations in the location selection field when you use it in the future.

Editing a location

If you edit a location, all boxs assigned to that location will have their location set with the changes location name. For instance, if you have a location called "Cellar" and you edit its name to be "Basement", all boxs that previously were assigned to the "Cellar" location will now be assigned to the "Basement" location.

Deleting a location

If you delete a location, all boxs assigned to that location will have their location set to "Uncategorized."
