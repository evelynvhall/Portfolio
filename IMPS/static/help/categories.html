Using Categories

About categories

IMPS categories are arbitrary groups that you can assign to items which provide an additional tool to further organize, sort, and find them. By default, new items are assigned to the "Uncategorized" category.

Assigning a category is optional. When you add or edit an item, you can:

  • Choose not to select a category, which means the item will saved as "Uncategorized"
  • Select an existing category to assign the item to
  • Type a new category name for this item. The new category will be used for this item and will be added to the list of categories in the category selection field when you use it in the future.

Managing categories

The easiest way to work with categories is to select or add them when adding items. However, you can add, edit, or delete categories using the control panel.

Adding a category

If you add a new category using the control panel, it will be added to the list of categories in the category selection field when you use it in the future.

Editing a category

If you edit a category, all items assigned to that category will have their category set with the changes category name. For instance, if you have a category called "decorating" and you edit its name to be "decorate", all items that previously were assigned to the "decorating" category will now be assigned to the "decorate" category.

Deleting a category

If you delete a category, all items assigned to that category will have their category set to "Uncategorized.
