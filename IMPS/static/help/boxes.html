Managing Boxes

IMPS uses the concept of boxes as containers for items. You must create a box before adding an item. Each box is assigned a number. IMPS will suggest the next available number sequentially or allow you to choose an unused number. Boxes may also be assigned a name and location, both of which are optional.

Adding a box

  1. From the home page, select Add box to begin adding a new box.
  2. IMPS will show the number of the first unused box (sequentially).
    • You may use this box number by selecting First available unused box number.
    • You may choose a different box number by selecting Choose number and entering a number. Box numbers must be positive integers.
  3. Enter a Location. You may select a location from the dropdown, type a new location name, or choose not to enter a location. If you do not choose a location, the box location will be automatically set to Unspecified.
  4. Enter a Box name. Box names are optional, but are printed on box labels and can help when searching for items.
  5. Click Add box when done to add the box to the database.

Viewing Boxes

  1. From the home page, select Box list to show the list of all boxes or By loc to select a location to show a list of boxes in a specific location.
  2. From the box list page, you can see information about the boxes currently in the IMPS database. On this page you can:
    • Click the items icon to see a list of all items in that box.
    • Click the edit icon to edit the box info.
    • Click the delete icon to delete the box.
    • Click the QR code icon to print the box label.

Edit a box

  1. From the home page, select Edit box to begin editing a box.
  2. Select a box number to edit.
  3. Enter a location. You may select a location from the dropdown or type a new location name. You may also delete the current location before submitting the changes. If you do remove the current location, the box location will be automatically set to Unspecified.
  4. Enter a Box name. Box names are optional, but are printed on box labels and can help when searching for items.
  5. Click Update when done to change the box info in the database.

Delete a box

  1. From the home page, select Del box to begin deleting a box.
  2. Select a box number to delete.
  3. Choose the action for any items in the box:
    1. Delete the box only, do nothing with the items
    2. Delete the box and move all items to another box
    3. Delete the box and delete all items assigned to the box
  4. Depending on your selection, you will need to confirm the action or select a new box that the items should be assigned to.
