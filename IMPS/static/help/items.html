Working with Items

IMPS helps you keep an inventory of items. Items must be placed into boxes so you must have created at least one box to start adding items to the inventory.

You can add items, edit items, find items, delete items, or move them.

Add an item

  1. From the home page, select Add item to begin adding a new item.
  2. Enter an item name. Each item must have a name.
  3. Enter and item category. You may select a category from the dropdown, type a new category name, or choose not to enter a category. If you type a new category, it will be added to the category list for you to choose from in the future. If you do not choose a category, the item's category will be automatically set to uncategorized.
  4. Enter a Box number. A box number is required. IMPS will check if the box number you enter is valid and will prompt for a different number if the one you enter does not exist in the database.
  5. Choose a photo (optional)
  6. Click Add ()when done to add the item to the database.
  7. IMPS will show the item detail page for you to verify that the item has been added correctly.

Edit an item

  1. Locate the item to begin editing it. You can find an item using the search tool, by browsing the list of all items (the inventory), by showing all the items in a particular category, or by showing all the items in a specific box.
  2. In the mobile view, click on the item and then the edit button ()to start the editing process. In the desktop view, click the edit icon to start editing.
  3. From the edit page, you can:
    • Change the item name. Each item must have a name.
    • Change and item category. You may select a category from the dropdown or type a new category name. if you delete the text from the category field, the item's category will be automatically set to uncategorized.
    • Change the Box number. A box number is required. The IMPS will check if the box number you enter is valid and will prompt for a different number if the one you enter does not exist in the database.
    • Click Update ()when done to update the item to the database.
  4. IMPS will show the item detail page for you to verify that the item has been added correctly.

Find an item

IMPS provides multiple ways to locate an item.

  1. Search: From the home page, click the search tool. Enter a search term in the search box, then press return or click the search button. The search tool will search item names, descriptions, and categories to find items that match the search term.
  2. Inventory: The inventory view shows a list of all your items, sorted with the most recently at the top. You can sort this list by name, date, category, or box.
  3. By category: From the home page, click the By category tool. Select a category from the dropdown, then press return or click the category button. IMPS will show a list of all items in the selected category.
  4. By box: From the home page, click the By box tool. Enter a box number then press return or click the box button. IMPS will show a list of all items in the selected box.

Delete an item

  1. Locate the item to be deleted. You can find an item using the search tool, by browsing the list of all items (the inventory), by showing all the items in a particular category, or by showing all the items in a specific box.
  2. In the mobile view, click on the item and then the Delete button to delete the item In the desktop view, click the trash can icon to delete.
  3. IMPS will show the delete confirmation page. You can either proceed to delete the item or cancel.

Move an item

  1. Locate the item to move. You can find an item using the search tool, by browsing the list of all items (the inventory), by showing all the items in a particular category, or by showing all the items in a specific box.
  2. In the mobile view, click on the item to open the item detail page. From here, you can update the item's box number to reflect its new location. In the desktop view, click the edit icon to open the item detail page.
